			\begin{block}{1. Introduction / Motivation}
				Write the problem in 3--6 lines. Posters should be readable quickly:
				\begin{itemize}
					\item What is the main goal?
					\item Why is it important?
					\item What is the challenge?
				\end{itemize}
				Example claim: we study the relationship $y = ax + b$ and estimate $a,b$ from data.
			\end{block}

			\begin{exampleblock}{2. Example}
				This is a boxed example using \texttt{exampleblock}.

				A displayed equation:
				\[
				\hat{a} = \arg\min_{a} \sum_{i=1}^{n} (y_i - ax_i)^2.
				\]
			\end{exampleblock}

			\begin{block}{3. Method (Simple Steps)}
				Keep the method short and visual:
				\begin{enumerate}
					\item Collect data $(x_i,y_i)$.
					\item Fit parameters by minimizing loss.
					\item Evaluate on a test set.
				\end{enumerate}
